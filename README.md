# Receipt Processor

This project is an AWS Lambda receipt processing app built to automate receipt handling.

## What it does
- uploads receipt files
- processes them with AWS Lambda
- uses AWS cloud services for automation

## Tools used
- AWS Lambda
- Amazon S3
- Python

## Purpose
I built this project to help my mom manage and organize receipts for her clients at work and for everyday expenses in daily life. I wanted to create something practical that could reduce manual work and make receipt processing easier. At the same time, this project helped me learn how AWS serverless services like Lambda and S3 can be used to automate real-world tasks.
